A review of objects in JavaScript

    Objects

  • Describe a thing
  • Are composed of properties and methods
  • Properties-
      nouns- or things the object is.
  • Methods-
      verbs- or things the object are.
  • Example: Dog
    var myDog={
      name: "krog",
      height: 10,
      color: "brown"
    }

    • Adding Methods to Objects

  • JavaScript objects can contain functions, just like they can contain variables.
  • a function inside of an object will be referred to as a method.
  • var myDog={
      name: "krog",
      height: 10,
      color: "brown",
      bark: function() {
      console.log("Oof oof"),
      }
    }

    • Invoking a method from an object

  • Use the dot operator to access the method and inoke it
  • var myDog={
      name: "krog",
      height: 10,
      color: "brown",
      bark: function() {
      console.log("Oof oof"),
      }
    }
    myDog.bark();

    • The 'this' keyword

  • Inside an object, the 'this' keyword refers to the object that the method resides in.
  • It allows you to access the other 'methods' and 'oroperties' in the object.
  • var myDog={
      name: "krog",
      height: 10,
      color: "brown",
      bark: function() {
      console.log(this.name+ "barks!"),
      }
    }
    myDog.bark();

    • Classes in JavaScript

  • Represent 'blueprints' of objects
  • They are abstract representations of objects
  • Idea of Dog -> Class
  • Actual Dog -> Object
